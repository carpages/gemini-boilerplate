# Gemini Boilerplate

[![Greenkeeper badge](https://badges.greenkeeper.io/carpages/gemini-boilerplate.svg)](https://greenkeeper.io/)

## Install

```
git clone https://github.com/carpages/gemini-boilerplate.git
npm i
bower i
```

## Use

### Development

```
grunt
```

### Production

```
grunt build
```
